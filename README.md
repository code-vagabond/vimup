# vimup
Vim config

Copy `.vimrc` to `~/.vimrc`

Install vim-gnome for system clipboard access

```bash
sudo apt-get install vim-gnome 
```
Download Vundle

```git
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

Install configured bundles
```git
 :BundleInstall
 ```
